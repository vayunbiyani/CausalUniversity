Causal University

This is where you can learn the ins-and-outs of Causal and financial modelling.

Causal's functionality goes relatively deep, letting you build extremely complex models for all kinds of use-cases. This comes with some nuances and intricacies that it's worth learning about via the documentation below.
👉 Start Here 📖 Articles 🎦 Tutorial Videos 📊 Example Models 📷 Webinars

Quick Start

  1. Watch the 4-minute tutorial video on the left. We guarantee that it's much faster than trying to figure things out for yourself.
  2. Go to the Formulas article — watch the video and skim the documentation.
  3. Watch the videos on Charts and Tables to learn how to visualise your model.
  4. You're now in a great position to start building your own model — try it out!
  5. When you're comfortable with the basics, learn the more advanced concepts:
    1. Categories
    2. Linking to models
    3. Connecting to data
  6. The rest of the concepts should be fairly self-explanatory, and you can treat the documentation as a reference as required.
